Terminal Commands

  • pwd – Show current directory structure.
  • date – Show current date on system.
  • cd | ../ | - | ../../ – Move through folders. Move 1 folder back. Move 2 folders back.
  • ls | -a | -al – List Directories. List All including hidden. List All Files with file permissions.
  • start . – Open GUI of Folder.
  • start – Open GUI file.
  • curl -0 – Saves a file from the web to your local folder you are in.
  • echo – Output on the next line.
  • history – See history.
  • touch – Create new file.
  • mkdir – Create directory
  • rm – Delete file.
  • rmdir – Delete Empty Directory Only
  • rm -r | -rf – Delete Directory and all its content including files Force Delete Directory and all its content including files. (Never use -rf unless you have to)
  • cp – Copy file.
  • cp -r – Copy directory content to new location.
  • mv – Move Items Files and Directories
  • unzip – Unzip file.
  • cat – Prints to screen what is in the file.
  • clear – Clear screen.
  • exit – Exit Terminal.
  • q – Exit file.
  • ping 10.1.1.1 – Check to see if you are connected and your computer is pinging to the internet.
